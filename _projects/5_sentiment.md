## Overview

Implementation of **sentiment classification** on the IMDB movie reviews dataset using traditional NLP techniques and machine learning.

## Approach

### Text Preprocessing
- Tokenization and lowercasing
- Stop word removal
- Stemming/Lemmatization

### Feature Extraction
- Bag of Words (BoW)
- TF-IDF vectorization
- N-gram features

### Classification Models
- Logistic Regression
- Naive Bayes
- Support Vector Machines

## Analysis

- Comparative analysis of different vectorization techniques
- Hyperparameter tuning for optimal performance
- Error analysis on misclassified reviews

## Technologies

`Python` `Scikit-learn` `NLTK` `Pandas` `Jupyter Notebook`
